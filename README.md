### **macOS Sequoia (15)**

Abrir um terminal (shell) do zsh e executar o comando:

    zsh -c "$(curl -fsSL https://raw.githubusercontent.com/fahadkalil/bigdata_installers/refs/heads/main/macos/installer.sh)"
